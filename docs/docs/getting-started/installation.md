# Installation

## Requirements

- Rust toolchain (for building from source)
- Or: pre-built binaries (coming soon)

## From Source

```bash
# Clone the repository
git clone https://github.com/weavster-dev/weavster.git
cd weavster

# Build and install
cargo install --path crates/weavster-cli
```

## Verify Installation

```bash
weavster --version
```
